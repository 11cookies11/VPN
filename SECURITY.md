# Security Policy

Language: English | [简体中文](SECURITY.zh-CN.md)

## Supported Versions

If this repository publishes releases, security fixes are applied to the latest
minor release line unless stated otherwise.

## Reporting a Vulnerability

Please report security issues privately.

- Preferred contact: <SECURITY_CONTACT>
- What to include: steps to reproduce, impact, affected versions, and any PoC

We will acknowledge receipt as soon as possible and coordinate a fix and release.
